# Publish Checklist

## Safe Private Prep

- [x] Product brief written.
- [x] Actor contract drafted.
- [x] Local extractor engine created.
- [x] Unit tests added.
- [x] Sample input and output generated.
- [x] Add marketplace listing copy and pricing draft.
- [x] Add screenshot or terminal demo capture.
- [ ] Add Apify `INPUT_SCHEMA.json`.
- [ ] Add Apify actor wrapper/package metadata.
- [ ] Package and test Apify runtime entrypoint.

## Publishing Blockers

- Apify publishing requires an accessible logged-in account/session and no hard-stop prompts.
- Do not enter payment, payout, legal, tax, ID, OTP/MFA, CAPTCHA, or account-security details.
